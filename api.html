src.client_template.client_method_template_f(method_name='', http_verb='', api_path='', doc_string='', path_params='')[source]¶
one size fits most method template
Parameters:
  • http_verb – GET, POST, PUT, DELETE and PATCH
  • method_name – a valid python function name as a string
  • api_path – a valid URL part which is joined with the BASE_PATH, can contain parameters. Wil be evaluated to a string.
  • doc_string – some description of the function and or endpoint
  • path_params – e.g. pagination is frequently used in the path
Returns:

string, ready to be appended to python-client-module
